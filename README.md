## ğŸ“ Project Description

The **College Event Management Portal** simplifies the entire event management process by centralizing all event-related information into a single platform. Students can explore categorized events such as **Technical**, **Cultural**, and **Sports**, view details, and directly register through official Instagram links.

Organizers get access to a dedicated panel where they can manage events, update event descriptions, and track attendance. The project focuses on providing a **clean UI**, **easy navigation**, and **real-world usability**.
